## Thino Embed | Thino 嵌入

`daily` can be true for current daily note, or a specific date like `2024/03/22 01:01:01`.

> [!note]- 中文
> - `daily` 可以是 true 表示当前每日笔记，或者是一个特定的日期，比如 `2024/03/22 01:01:01`。

````markdown
```thino
{
  list: true,
  calendar: true,
  heatmap: true,
  status: true,
  editor: true,
  daily: '2024/03/22 01:01:01', 
}
```
````
